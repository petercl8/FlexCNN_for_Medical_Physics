Dataset
=======
1) Original dataset is of size: 382x513 (radial distance x angular sampling).
    Training set: 17830 examples
    Validation set: 3653 examples
    Test set: 5729 examples


2) 320x257: Here, vertical size is left alone. The horizontal size has been replication padded by 1 and then average pooled to be of horizontal size 257. These were used for the 288x288 experiment which used cropping/pooling to resize.

3) 288x257: Here, vertical size was cropped to be of size 288 and the horizontal was resized using bilinear interpolation. This was used to compare with the 288x288 experiment which used cropping/pooling.

4) All other types: These have had initial vertical cropping to size 320. Then, they have been bilinearly resized to be of size NxN. To use these datasets, set sino_init_vert_cut = 0 (since these were already vertically cropped before resizing).
